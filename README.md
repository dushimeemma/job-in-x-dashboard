# Job In X | Analytics

Jobs adverts dashboard

## Vision

Provide figures for adverts on different websites

## Prerequisites

- node
- npm | yarn
- job in x API

## Getting Started

- clone the repo
- install dependencies `yarn | npm install`
- run the application in local environment `yarn dev | npm run dev`
